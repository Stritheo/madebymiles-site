# Miles Sowden | Executive Leader

Modern, high-performance professional website showcasing executive leadership in insurance, data and AI transformation.

**Live site:** https://madebymiles.ai

## About

Executive positioning site for Miles Sowden, GM Data and Technology Solutions at Strata Community Insurance.

Features 20 years of executive experience across:
- Strata Community Insurance
- Commonwealth Bank
- Hollard Insurance
- Westpac Banking Corporation
- Suncorp/Promina

## Features

### Interactive Career Timeline
- SVG-based timeline with hover states and tooltips
- Horizontal layout on desktop, vertical on mobile
- Full keyboard navigation and accessibility
- Click nodes to scroll to detailed project cards

### Modern Architecture
- **Next.js 14** with App Router
- **TypeScript** for type safety
- **Server Components** by default for performance
- **Container Queries** for responsive layouts
- **Design System** with CSS tokens and motion utilities

### Performance & Accessibility
- ✅ Optimized fonts with display: swap
- ✅ Semantic HTML structure
- ✅ ARIA labels and focus management
- ✅ Respects prefers-reduced-motion
- ✅ Server-side rendering for fast loads
- ✅ Zero layout shift (CLS)

### SEO Optimized
- JSON-LD structured data (Person + Organization)
- Open Graph and Twitter Card tags
- Dynamic sitemap.xml and robots.txt
- Comprehensive metadata

## Structure

- **Hero**: Executive positioning statement
- **What I do**: Core capabilities with container query grids
- **Career Timeline**: Interactive journey visualization
- **Work and impact**: Detailed achievements with metrics
- **Advisory**: Services and opportunities

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deployment

### Vercel (Recommended)

1. Push to GitHub
2. Import repository in Vercel
3. Deploy automatically

- Node.js 18+
- Next.js 14
