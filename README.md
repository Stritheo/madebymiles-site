# Miles Sowden | Executive Leader

Professional website showcasing executive leadership in insurance, data and AI transformation.

**Live site:** https://madebymiles.ai

## About

Executive positioning site for Miles Sowden, GM Data and Technology Solutions at Strata Community Insurance. 

Features executive experience across:
- Strata Community Insurance
- Commonwealth Bank
- Hollard Insurance  
- Westpac Banking Corporation
- Suncorp/Promina

## Technical Details

Single-page HTML site optimised for:
- Professional presentation to boards and executive search firms
- Fast loading and accessibility
- Clean typography and executive presence
- Mobile responsive design

## Structure

- **Hero**: Executive positioning statement
- **What I do**: Core capabilities (4 areas)
- **Work and impact**: Major achievements with metrics
- **Advisory**: Services and opportunities

## Updates

To update content, edit `index.html` and commit to main branch. GitHub Pages deploys automatically.
